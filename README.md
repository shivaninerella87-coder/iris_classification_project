# 🌸 Iris Data Classification using AI

## 📌 Overview
This project is a Machine Learning application that classifies Iris flowers into different species using Artificial Intelligence. The model is trained on the Iris dataset provided by Scikit-Learn and predicts the flower species based on user-provided measurements.

---

## 🚀 Features
* Train a Machine Learning classification model
* Predict Iris flower species
* Uses the Scikit-Learn Iris dataset
* Saves the trained model using Joblib
* Simple command-line interface
* Beginner-friendly AI project

---

## 🛠️ Technologies Used
* Python
* Scikit-Learn
* Joblib
* NumPy

⚙️ Installation
1. Clone the repository:
git clone [https://github.com/shivaninerella87-coder/iris_classification_project.git](https://github.com/shivaninerella87-coder/iris_classification_project.git)

2. Move into the project folder:
cd iris_classification_project

3. Install the required packages:
pip install -r requirements.txt

▶️ Usage
Run the main pipeline program:
python main.py

Example Output Scenario:

Sepal Length: 5.4
Sepal Width: 3.8
Petal Length: 2.9
Petal Width: 1.2

Predicted Flower: Versicolor

📊 Dataset
This project uses the famous Iris dataset directly available within the Scikit-Learn library, ensuring easy accessibility and balanced data for model training.
