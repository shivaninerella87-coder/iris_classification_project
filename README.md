# iris_classification_project
README
🌸 Data Classification using AI
📌 Overview
This project is a simple Machine Learning application that classifies Iris flowers into different species using Artificial Intelligence (Machine Learning). The model is trained on the Iris dataset provided by Scikit-Learn and predicts the flower species based on user-provided measurements.

🚀 Features
. Train a Machine Learning classification model
. Predict Iris flower species
. Uses the Scikit-Learn Iris dataset
. Saves the trained model using Joblib
. Simple command-line interface
. Beginner-friendly AI project

🛠️ Technologies Used
. Python
. Scikit-Learn
. Joblib
. NumPy

📂 Project Structure
iris_classification_project/
│
├── src/
│   ├── __init__.py                 # Keeps the folder recognizable as a Python package
│   ├── data_preprocessing.py       # Handles data loading, splitting, and scaling
│   └── model_training.py           # Handles the KNN algorithm and evaluation metrics
│
├── main.py                         # The central script to run the entire pipeline
└── requirements.txt                # Lists the required Python libraries

⚙️ Installation
Clone the repository:

git clone https://github.com/shivaninerella87-coder/iris_classification_project
Move into the project folder:

cd DataClassification
Install the required packages:

pip install -r requirements.txt

▶️ Usage
Train the model:

python train_model.py
Run the prediction program:

python predict.py

Enter the flower measurements when prompted.

Example:

Sepal Length: 5.4
Sepal Width: 3.8
Petal Length: 2.9
Petal Width: 1.2
Output:

Predicted Flower: Versicolor

📊 Dataset
This project uses the Iris dataset available in the Scikit-Learn library.
